### 进入后端文件夹

```
cd shop-backend
```

### 后端安装依赖
```
npm install      
```

### 启动后端服务
```
npm start
```

### 进入前端文件夹
```
cd Mobile-terminal-mall
```

### 安装依赖：

```
yarn
```

### 启动服务
```
yarn dev
```
